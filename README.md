# Pornhub-Video-Downloader
Pornhub视频下载Chrome插件

通过向Pornhub视频页面注入JS获取视频接口信息，提取链接下载。

## 2021年1月13日 更新，感谢大家的支持！
1.如果是Chrome (谷歌)浏览器，下载zip后解压加载
Pornhub视频下载插件v1.9.zip (点这里下载)

本地加载插件的方法：

url地址栏打开    chrome://extensions/
